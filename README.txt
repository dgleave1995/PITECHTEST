Requirements:

- Node.js: ^20.19.0 or >=22.12.0
- Vue: ^3.5.18
- Vite: ^7.0.6
- Pinia: ^3.0.3
- Vue Router: ^4.5.1

Dev Tools:

- Vitest: ^3.2.4 (unit tests)  

Installation:

git clone https://github.com/dgleave1995/PITechTest.git
cd PITechTest
npm install

Install Dependencies:

npm install

Run Development Server:

npm run dev

Run Unit Tests:

npm run test:unit
